CV React – Actividad de Renderizado Condicional y Renderizado de Listas
Descripción del Proyecto

Este proyecto forma parte de la actividad práctica del módulo de Desarrollo Web con React.
El objetivo es construir una interfaz dinámica para un Currículum Vitae (CV) utilizando:

Renderizado condicional

Renderizado de listas (map)

Componentes funcionales

Estilos dinámicos según datos del usuario

Incluye las secciones:

Experiencia Laboral

Formación Académica

Stack de Tecnologías (con etiquetas que cambian de color según la tecnología)

Además, todo el desarrollo fue gestionado con Git y GitHub, utilizando commits descriptivos para reflejar cada avance.

Tecnologías utilizadas

React + Vite

JavaScript

CSS

Node.js

Git / GitHub

Características principales
✔️ Componente Experiencia

Array dinámico con mínimo 10 experiencias o proyectos

Renderización con .map()

Diseño limpio y legible

✔️ Componente Educación

Lista de cursos, técnicos, tecnólogo o formación complementaria

Datos generados desde un arreglo de objetos

✔️ Componente StackTecnologías

Renderizado condicional para asignar colores según la tecnología

Implementación fiel al ejemplo del tutorial

Instalación y ejecución del proyecto
1. Clonar el repositorio
git clone https://github.com/DESARROLLO-WEB-OROZCO/CV-REACT-DAVID-OROZCO-2.git

2. Entrar al proyecto
cd CV-REACT-DAVID-OROZCO-2

3. Instalar dependencias
npm install

4. Ejecutar el servidor de desarrollo
npm run dev

5. Abrir el proyecto en el navegador

Generalmente en:

http://localhost:5173
